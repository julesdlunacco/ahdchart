# AHD Charts

AHD Charts is a WordPress plugin that generates Human Design charts with a rich, configurable front-end UI and an admin settings page for theme customization. It renders a precise Human Design bodygraph, planet information panels, and a full textual report.

- Front‑end React app for chart input, saved charts, and results
- SVG bodygraph with configurable theming (centers, gates, active gate circles, etc.)
- Planet activation side panels and type/authority/profile/definition summary boxes
- Admin settings app for live theme preview and presets
- Export tools for PNG/PDF of the chart and text/markdown of the full report
